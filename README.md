# 待办清单桌面应用

> Python + tkinter + SQLite 实现的桌面待办事项管理软件

## 项目简介

本项目是一个使用 Python 开发的桌面待办清单应用，采用 tkinter 构建图形界面，SQLite 进行数据持久化存储。

## 功能特性

- ✅ **任务管理**：添加、编辑、删除待办事项
- ✅ **分类管理**：支持自定义分类，可为每个分类设置颜色标识
- ✅ **优先级设置**：支持高、中、低三级优先级
- ✅ **筛选搜索**：按分类、完成状态筛选，支持关键词搜索
- ✅ **截止日期**：可为任务设置截止日期
- ✅ **状态统计**：实时显示任务完成统计信息
- ✅ **数据持久化**：使用 SQLite 本地存储，数据自动保存

## 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| GUI 框架 | tkinter | Python 内置图形库 |
| 数据库 | sqlite3 | Python 内置轻量级数据库 |
| 开发语言 | Python 3 | 纯 Python 实现，无需额外依赖 |

## 运行方式

### 环境要求
- Python 3.8 或更高版本

### 启动应用

```bash
cd todo-app
python main.py
```

> 注：本项目仅使用 Python 内置模块（tkinter、sqlite3），无需安装额外依赖。

## 使用说明

1. **添加任务**：点击顶部 "+ 新建待办" 按钮
2. **编辑任务**：双击任务项，或选中后点击"编辑"按钮
3. **完成任务**：选中任务后点击"标记完成/未完成"
4. **删除任务**：选中任务后点击"删除"按钮
5. **管理分类**：点击"管理分类"按钮添加/删除分类
6. **筛选任务**：使用左侧分类列表或顶部筛选条件
7. **搜索任务**：在搜索框输入关键词，点击"搜索"

## 数据存储

应用数据存储在同级目录下的 `todo.db` 文件中，首次运行会自动创建数据库和默认分类。
